###< Installation >
To install and run this project, follow the below steps:

####1. Clone Repo
`git clone https://github.com/soltran/coinmo

####2. Install Cordova Ionic  
`npm install -g cordova ionic`

####3. Npm Install components
`npm install`

####4. Start Ionic Server
`ionic serve`
